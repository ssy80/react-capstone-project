For reviewing this project.

1) download the app zip file from github.
2) unzip the folder to your computer.
3) create a new react app by command npx create-react-app react-review-app.
4) copy all the files/folders from the unzipped folder into the react-review-app folder.
5) open terminal use command "npm audit fix" to install missing dependency modules to the current react app.
6) start the app "npm start"
